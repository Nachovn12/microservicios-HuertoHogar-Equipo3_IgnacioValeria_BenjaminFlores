# Usuario Microservicio

Microservicio para gestión de usuarios del sistema Huerto Hogar.

## Funcionalidades Planificadas
- Registro de usuarios
- Autenticación con JWT
- Gestión de perfiles de usuario
- Roles y permisos

## Tecnologías
- Spring Boot 3.1.5
- Spring Security
- JWT (JSON Web Tokens)
- MySQL 8.0
- Java 17
